Modèle Logique des Données

MLD

Un MLD peut être généré à partir d'un MCD. Il permettra de représenter schématiquement les tables à créer dans une base de données.

On ne parlera alors plus d'entités mais de tables, les identifiants deviennent des clés et les attributs des colonnes.

Pour effectuer la transition, nous allons appliquer un certain nombre de changements...

Passage du MCD au MLD

Entité A Entité B Résultat
0,1 ou 1,1 0,N ou 1,N 1-N : Création d'une clé étrangère dans l'entité A
0,N ou 1,N 0,N ou 1,N N-N : La relation entre les entités est transformée en table
Se focaliser sur le symbole après la virgule

Exercice

Réalisez le MCD puis le MLD d'une application de gestion d'une école en suivant les règles de gestion suivantes :

  1. Un étudiant peut être dans une classe différente chaque année
  2. Une classe peut assister à plusieurs cours dans l'année mais il ne peut y avoir qu'une seule classe par cours
  3. Un enseignant peut donner plusieurs cours dans l'année mais il ne peut y avoir qu'un seul enseignant par cours

Exercice

Réalisez le MCD puis le MLD d'un site ecommerce en suivant les règles de gestion suivantes :

  1. Un internaute peut créer un compte et passer des commandes
  2. Chaque commande est associée à un moyen de paiement
  3. Une commande contient un ou plusieurs produits
  4. Un même produit peut être commandé plusieurs fois
